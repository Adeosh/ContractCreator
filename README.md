# 📄 ContractCreator (ERP-система управления договорами)

ContractCreator — это современное кроссплатформенное десктопное приложение для автоматизации документооборота, ведения реестра контрагентов и управления жизненным циклом договоров.
Проект разработан с упором на производительность, строгую типизацию и масштабируемость, используя передовые подходы экосистемы .NET.

## ✨ Ключевые возможности

* 🏢 Мультифирменность: Управление несколькими собственными организациями (ООО, ИП) в едином пространстве.
* 🤝 Реестр контрагентов: Хранение реквизитов (ИНН, КПП, ОГРН), контактных лиц, адресов и банковских счетов.
* 📝 Управление договорами:
  * Отслеживание стадий и сроков (тендер, исполнение, расторжение).
  * Ведение спецификаций (номенклатура, цены, НДС).
  * Разбиение контрактов на этапы выполнения.
* 🗂 Генерация документов: Автоматическое создание связанных Актов, Счетов на оплату и Накладных (УПД) на основе спецификации договора.
* 📎 Файловое хранилище: Прикрепление сканов, печатей (факсимиле) и документов к карточкам фирм и контрактов.

## 🛠 Технологический стек

* Платформа: C# / .NET 10
* Пользовательский интерфейс: [Avalonia UI] (XAML) с поддержкой темной/светлой темы.
* Паттерн проектирования UI: MVVM (Model-View-ViewModel) с использованием `ReactiveUI`.
* Архитектура: Clean Architecture (Чистая архитектура).
* База данных / ORM: PostgreSQL, Entity Framework Core.
* Логирование: Serilog.

## 🏗 Архитектура проекта (Clean Architecture)

Решение разделено на слабосвязанные слои для обеспечения тестируемости и простоты поддержки:

1. `ContractCreator.Domain` — Сердце системы. Содержит бизнес-сущности, интерфейсы репозиториев и value objects.
2. `ContractCreator.Application` — Прикладная логика, сервисы, обработчики команд.
3. `ContractCreator.Infrastructure` — Работа с базой данных (EF Core DbContext, миграции), файловой системой и внешними API.
4. `ContractCreator.Shared` — DTO (Data Transfer Objects) и Enum'ы, общие для всего приложения.
5. `ContractCreator.UI` — Слой представления на Avalonia. Содержит Views, ViewModels, конвертеры, стили и Behaviors.
6. `ContractCreator.Tests.*` — Модульные (Unit) и интеграционные (Integration) тесты.
